O(N²) LOOP VISUALIZER

  
A 3D visualization tool that teaches algorithm time complexity. Watch nested loops execute as cubes light up in a grid — making "N times N" tangible.

  
HOW TO USE

  
PLAYStart the animation — cubes light up as each (i, j) iteration runs

  
STEPAdvance one iteration at a time for studying each step

  
EDITClick the EDIT badge on the code panel to modify loop bounds live

  
DRAGDrag to orbit the camera, scroll to zoom, right-drag to pan

  
CLICK CUBEClick any cube in the grid to see its (i, j) position and state

  
FEATURES

  
HEATColors cubes blue→red by visit order — see the sweep pattern

  
ROW / COLToggle between row-major and column-major traversal

  
CMPCompare O(N) vs O(N²) vs O(N³) — see "What if N was bigger?"

  
SFXProcedural sound effects — pitch rises with progress

  
TOPBird's-eye view to see the full grid pattern

  
DEBUGGERCode panel shows live values, conditions, and execution pointer during animation

  
KEYBOARD

  
Space: play/pause   S: step   R: reset   T: top view   M: mute   ↑↓: N size   ←→: speed   ?: shortcuts   I: this overlay
